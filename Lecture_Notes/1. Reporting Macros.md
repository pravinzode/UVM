# Display and uvm_info Differnce 
- default color for display in is **black** 
- uvm_info : 
	- informative message -*blue*, 
	- Warning -*yellow*,
	- Error - *red*
	- Fatal - *Magenta-*
	- Debug - *Cyan*
	- Reset - *Default*
- ID is visible,reporting macro ,Time, line number, file name in reporting macros

- Various Reporting macros are 
	- UVM_INFO - UVM_DISPLAY
	- UVM_WARNING- UVM_DISPLAY
	- UVM_ERROR - UVM_DISPAY , UVM_COUNT
	- UVM_FATAL - UVM_DISPLAY , UVM_EXIT
- UVM Macros semicolon can be avoided 
- Demonstrate the macro with diffent methods
